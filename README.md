# Z80Compiler
Java command line application that compiles Java-like programs to Z80 assembler and binary.

Development:
All you need is Java JDK. More is better.
* javac *.java
* java RegressionTest

Usage:
* java Z80Compiler
* java Z80Compiler -r test1.j
* java Z80Compiler -r -z -b -d test1.j >test1.log
